# Steps to setup project

1. Clone the project on your local by 

	`git clone`

2. Create database on local

	`rake db:create`

3. Migrate all the tables

	`rake db:migrate`

4. Insert dummy records in tables (Optional)
	
	`rake db:seed`

5. Start the server

	`rails server`

6. Visit the URL `http://localhost:3000`
